# Best practices

## Introduction

In this project I apply two basic best practices like unit test and using makefile


## Folder Structure
```bash
├── Makefile
├── Pipfile
├── Pipfile.lock
├── README.md
├── model.py
└── tests
    ├── __init__.py
    └── model_test.py
```


## Set up
You just need to run the makefile to run unittest

```bash
make setup
```

```bash
make test
```
